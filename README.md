# 🎵 بوت الأناشيد — Telegram Music Bot

بوت تيلجرام يبحث عن الأناشيد والأغاني في يوتيوب ويرسلها للمستخدم صوتاً أو فيديو.

---

## ✨ المميزات

- 🔍 **بحث ذكي** — يجلب 20 نتيجة من يوتيوب عند كتابة اسم النشيد
- 📄 **صفحات التنقل** — تعرض 5 نتائج في كل صفحة مع أزرار التالي/السابق
- 🎧 **تحميل صوت** — يستخرج الصوت بصيغة MP3 بجودة 192kbps
- 🎬 **تحميل فيديو** — يرسل الفيديو بصيغة MP4 بأفضل جودة متاحة
- ♻️ **تنظيف تلقائي** — يحذف الملفات المؤقتة بعد الإرسال

---

## 🚀 التشغيل على Railway

### 1. المتطلبات
- حساب [Railway](https://railway.app)
- توكن بوت تيلجرام من [@BotFather](https://t.me/BotFather)

### 2. النشر
1. ارفع المستودع على GitHub
2. في Railway: **New Project → Deploy from GitHub repo**
3. اختر هذا المستودع
4. أضف متغير البيئة:

```
TELEGRAM_BOT_TOKEN=your_token_here
```

5. Railway سيبني الـ Docker تلقائياً وسيشغّل البوت

### 3. متغيرات البيئة

| المتغير | الوصف | مطلوب |
|---------|-------|--------|
| `TELEGRAM_BOT_TOKEN` | توكن البوت من @BotFather | ✅ نعم |

---

## 🛠️ التشغيل محلياً

```bash
# تثبيت المتطلبات
pip install -r requirements.txt

# تعيين التوكن
export TELEGRAM_BOT_TOKEN=your_token_here

# تشغيل البوت
python bot.py
```

> **ملاحظة:** يجب تثبيت `ffmpeg` على النظام لاستخراج الصوت.
> - Linux: `apt install ffmpeg`
> - Mac: `brew install ffmpeg`
> - Windows: [تحميل من الموقع الرسمي](https://ffmpeg.org/download.html)
